Lyrics Wild, es un Linux Widget de letras sincronizadas para Linux que.

Hace todo: busca la letra de la canción en distintas páginas que ofrecen el archivo “.lrc” (lyrics con los tiempos). Si no los encuentra, también busca en YouTube, versiones con subtítulos. Y si aún así no los encuentra, incluye una opción bien artesanal que permite convertir texto en lyrics sincronizado: puede buscar la letra o puedes ingresarla manualmente, y luego mientras la canción suena, solo haces clic en el momento que suena cada frase, y se van guardando con los tiempos correctos, para después poder escuchar con subtítulos siempre sincronizados.

Además cantidad de funciones, como:

• Cambiar las Pieles, y modificar la estética.

• Traducir, y mostrar traducción debajo de la linea.

• También:

• Mostrar siempre encima (o no).

• Clickeable a través (o no).

• Redimensionar, darle la forma y el tamaño que se desea.

• Cambiar la opacidad, del fondo.

• Mostrar reloj.

• Sincronizarse al segundo exacto que va la letra, para reparar si es de otras versiones.

• Usar o desactivar Skins para que mantener el color natural del tono del fondo de pantalla o del sistema.

• Texto con alto contraste.
